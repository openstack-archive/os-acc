============
Installation
============

At the command line:

.. code-block:: shell

   $ pip install os-acc

Or, if you have virtualenvwrapper installed:

.. code-block:: shell

   $ mkvirtualenv os-acc
   $ pip install os-acc

Or, from source:

.. code-block:: shell

   $ git clone https://github.com/openstack/os-acc
   $ cd os-acc
   $ python setup.py install
